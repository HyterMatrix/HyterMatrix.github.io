еңЁUbuntuдёҠжҢӮиҪҪиҝңзЁӢSMBж–Үд»¶еӨ№

2023/12/26 03:05 дёҠеҚҲ posted in  жңҚеҠЎеҷЁ
  • жҢӮиҪҪ
  • еҚёиҪҪ

жҢӮиҪҪ

еңЁUbuntuдёҠжҢӮиҪҪиҝңзЁӢSMBж–Үд»¶еӨ№зҡ„иҝҮзЁӢйҖҡеёёеҲҶдёәд»ҘдёӢеҮ дёӘжӯҘйӘӨпјҡ

  1. е®үиЈ…еҝ…иҰҒзҡ„иҪҜд»¶еҢ…пјҡ
    йҰ–е…ҲпјҢзЎ®дҝқе·Іе®үиЈ…cifs-utilsиҪҜд»¶еҢ…гҖӮиҝҷдёӘеҢ…жҸҗдҫӣдәҶжҢӮиҪҪSMBе…ұдә«жүҖйңҖзҡ„е·Ҙе…·гҖӮдҪ еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨжқҘе®үиЈ…е®ғпјҡ

    sudo apt-get update
sudo apt-get install cifs-utils

  2. еҲӣе»әжҢӮиҪҪзӮ№пјҡ
    жҺҘдёӢжқҘпјҢеҲӣе»әдёҖдёӘжң¬ең°зӣ®еҪ•жқҘдҪңдёәиҝңзЁӢSMBе…ұдә«зҡ„жҢӮиҪҪзӮ№гҖӮдҫӢеҰӮпјҡ

    sudo mkdir /mnt/myshare

  3. зј–иҫ‘fstabж–Үд»¶пјҲеҸҜйҖүпјүпјҡ
    еҰӮжһңдҪ еёҢжңӣеңЁжҜҸж¬ЎеҗҜеҠЁж—¶иҮӘеҠЁжҢӮиҪҪSMBе…ұдә«пјҢеҸҜд»Ҙзј–иҫ‘/etc/fstabж–Үд»¶гҖӮеңЁж–Үд»¶жң«е°ҫж·»еҠ дёҖиЎҢзұ»дјјд»ҘдёӢж јејҸзҡ„еҶ…е®№пјҡ

    //server/share /mnt/myshare cifs credentials=/path/to/credentials,file_mode=0777,dir_mode=0777 0 0

    е…¶дёӯпјҢserverжҳҜSMBжңҚеҠЎеҷЁзҡ„ең°еқҖпјҢshareжҳҜе…ұдә«зҡ„еҗҚз§°пјҢ/mnt/myshareжҳҜдҪ еҲӣе»әзҡ„жң¬ең°жҢӮиҪҪзӮ№пјҢ/path/to/credentialsжҳҜдёҖдёӘеҢ…еҗ«з”ЁжҲ·еҗҚе’ҢеҜҶз Ғзҡ„ж–Үд»¶и·Ҝеҫ„гҖӮ

  4. еҲӣе»әеҮӯиҜҒж–Үд»¶пјҲеҰӮжһңзј–иҫ‘дәҶfstabпјүпјҡ
    еҰӮжһңдҪ еҶіе®ҡзј–иҫ‘fstabж–Үд»¶пјҢйңҖиҰҒеҲӣе»әдёҖдёӘеҢ…еҗ«SMBзҷ»еҪ•еҮӯиҜҒзҡ„ж–Үд»¶гҖӮиҝҷдёӘж–Үд»¶йҖҡеёёеҢ…еҗ«д»ҘдёӢеҶ…е®№пјҡ

    username=your_username
password=your_password

    зЎ®дҝқиҝҷдёӘж–Үд»¶зҡ„жқғйҷҗи®ҫзҪ®еҫ—еҪ“пјҢд»ҘйҒҝе…Қе®үе…ЁйЈҺйҷ©гҖӮ

  5. жүӢеҠЁжҢӮиҪҪпјҲеҰӮжһңжІЎжңүзј–иҫ‘fstabпјүпјҡ
    еҰӮжһңдҪ дёҚеёҢжңӣе…ұдә«еңЁжҜҸж¬ЎеҗҜеҠЁж—¶иҮӘеҠЁжҢӮиҪҪпјҢеҸҜд»ҘжүӢеҠЁжҢӮиҪҪе®ғгҖӮдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

    sudo mount -t cifs -o username=your_username,password=your_password //server/share /mnt/myshare

  6. и®ҝй—®е’ҢдҪҝз”Ёе…ұдә«пјҡ
    дёҖж—Ұе…ұдә«иў«жҢӮиҪҪпјҢдҪ е°ұеҸҜд»ҘеғҸи®ҝй—®жң¬ең°ж–Үд»¶зі»з»ҹдёҖж ·и®ҝй—®е’ҢдҪҝз”Ёе®ғгҖӮ

еҚёиҪҪ

иҰҒеҸ–ж¶ҲжҢӮиҪҪпјҲеҚёиҪҪпјүдёҖдёӘе·Із»ҸжҢӮиҪҪзҡ„SMBе…ұдә«пјҢдҪ еҸҜд»ҘдҪҝз”Ёumountе‘Ҫд»ӨгҖӮиҝҷжҳҜдёҖдёӘж ҮеҮҶзҡ„е‘Ҫд»ӨпјҢз”ЁдәҺеҚёиҪҪд»»дҪ•зұ»еһӢзҡ„ж–Үд»¶зі»з»ҹпјҢеҢ…жӢ¬SMBе…ұдә«гҖӮд»ҘдёӢжҳҜе…·дҪ“жӯҘйӘӨпјҡ

  1. дҪҝз”Ёumountе‘Ҫд»Өпјҡ
    жү“ејҖз»Ҳз«ҜпјҢе№¶иҫ“е…Ҙд»ҘдёӢе‘Ҫд»ӨжқҘеҚёиҪҪSMBе…ұдә«пјҡ

    sudo umount /mnt/myshare

    е…¶дёӯ/mnt/myshareжҳҜдҪ д№ӢеүҚеҲӣе»әзҡ„жҢӮиҪҪзӮ№гҖӮ

  2. зЎ®и®ӨеҚёиҪҪжҲҗеҠҹпјҡ
    еҚёиҪҪе®ҢжҲҗеҗҺпјҢдҪ еҸҜд»ҘдҪҝз”Ёdf -hжҲ–mountе‘Ҫд»ӨжқҘжЈҖжҹҘе…ұдә«жҳҜеҗҰе·Із»Ҹиў«жҲҗеҠҹеҚёиҪҪгҖӮеҰӮжһңе…ұдә«дёҚеҶҚеҮәзҺ°еңЁеҲ—иЎЁдёӯпјҢиҝҷиЎЁзӨәе®ғе·Із»Ҹиў«еҚёиҪҪгҖӮ

  3. еӨ„зҗҶеҚёиҪҪй—®йўҳпјҡ

    • еҰӮжһңеңЁе°қиҜ•еҚёиҪҪж—¶йҒҮеҲ°вҖңи®ҫеӨҮжӯЈеҝҷвҖқзҡ„й”ҷиҜҜпјҢиҝҷйҖҡеёёж„Ҹе‘ізқҖжңүиҝӣзЁӢжӯЈеңЁдҪҝз”ЁжҢӮиҪҪзӮ№гҖӮзЎ®дҝқжІЎжңүд»»дҪ•иҝӣзЁӢпјҲеҰӮж–Үд»¶жөҸи§ҲеҷЁжҲ–е‘Ҫд»ӨиЎҢе·Ҙе…·пјүжӯЈеңЁи®ҝй—®иҜҘжҢӮиҪҪзӮ№пјҢз„¶еҗҺеҶҚж¬Ўе°қиҜ•еҚёиҪҪгҖӮ
    • еҰӮжһңд»Қз„¶ж— жі•еҚёиҪҪпјҢдҪ еҸҜд»ҘдҪҝз”Ёlsofе‘Ҫд»ӨжҹҘзңӢе“ӘдёӘиҝӣзЁӢжӯЈеңЁдҪҝз”ЁжҢӮиҪҪзӮ№пјҡ 
      sudo lsof /mnt/myshare
      з„¶еҗҺеҸҜд»ҘеҶіе®ҡжҳҜеҗҰе…ій—ӯзӣёе…іиҝӣзЁӢжҲ–з»Ҳжӯўзӣёе…ідјҡиҜқпјҢз„¶еҗҺеҶҚе°қиҜ•еҚёиҪҪгҖӮ

  4. иҮӘеҠЁеҚёиҪҪпјҲеҸҜйҖүпјүпјҡ
    еҰӮжһңдҪ зҡ„зі»з»ҹж”ҜжҢҒпјҢдҪ д№ҹеҸҜд»ҘдҪҝз”Ёumount -aе‘Ҫд»ӨеҚёиҪҪжүҖжңүе·ІжҢӮиҪҪзҡ„ж–Үд»¶зі»з»ҹгҖӮдҪҶиҜ·е°ҸеҝғдҪҝз”ЁиҝҷдёӘе‘Ҫд»ӨпјҢеӣ дёәе®ғдјҡе°қиҜ•еҚёиҪҪжүҖжңүжҢӮиҪҪзӮ№пјҢеҢ…жӢ¬зі»з»ҹиҮӘеҠЁжҢӮиҪҪзҡ„зӮ№гҖӮ

еңЁеӨ„зҗҶжҢӮиҪҪе’ҢеҚёиҪҪж“ҚдҪңж—¶пјҢиҜ·зЎ®дҝқдҪ дәҶи§Је‘Ҫд»Өзҡ„еҗ«д№үе’ҢеҸҜиғҪзҡ„еҪұе“ҚгҖӮй”ҷиҜҜзҡ„ж“ҚдҪңеҸҜиғҪдјҡеҜјиҮҙж•°жҚ®дёўеӨұжҲ–зі»з»ҹдёҚзЁіе®ҡгҖӮеҰӮжһңдҪ дёҚзЎ®е®ҡпјҢе»әи®®е’ЁиҜўз»ҸйӘҢдё°еҜҢзҡ„з”ЁжҲ·жҲ–з®ЎзҗҶе‘ҳгҖӮ
