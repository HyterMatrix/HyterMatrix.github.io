nvm install <version> [arch] пјҡе®үиЈ…nodeпјҢ versionжҳҜзүҲжң¬еҸ·д№ҹеҸҜд»ҘжҳҜlatestпјҲжңҖж–°зЁіе®ҡзүҲжң¬пјүгҖӮеҸҜйҖүеҸӮж•°archжҢҮзі»з»ҹдҪҚж•°64/32пјҢй»ҳи®ӨжҳҜеҪ“еүҚзі»з»ҹдҪҚж•°гҖӮ
 
nvm uninstall <version> пјҡеҚёиҪҪжҢҮе®ҡзүҲжң¬nodeгҖӮ
 
nvm list [available] пјҡжҳҫзӨәе·Іе®үиЈ…зҡ„еҲ—иЎЁпјҢеҸҜйҖүеҸӮж•°availableпјҢжҳҫзӨәеҸҜе®үиЈ…зҡ„жүҖжңүзүҲжң¬гҖӮlistеҸҜз®ҖеҢ–дёәls
 
nvm use [version] [arch] пјҡдҪҝз”ЁжҢҮе®ҡзүҲжң¬nodeпјҢеҸҜйҖүеҸӮж•°archеҸҜжҢҮе®ҡ32/64дҪҚгҖӮиҝҷдёӘжҳҜе…ЁеұҖзҡ„гҖӮ
